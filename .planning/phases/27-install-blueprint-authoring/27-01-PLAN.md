  when helmChart.repository starts with "oci://": chart_ref = repository.rstrip('/') + "/" + helmChart.name

HTTPS chart-ref assembly (operator main.py:1114-1116):
  repo_name = last path segment of repository; chart_ref = repo_name + "/" + name.
  So repository "https://weka.github.io/csi-wekafs" + name "csi-wekafsplugin" => "csi-wekafs/csi-wekafsplugin".

readinessCheck dispatcher (main.py wait_for_component_ready) supports type: pod | deployment | statefulset | job.
  type: deployment waits available >= desired. type: job waits for job completion.
