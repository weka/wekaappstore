## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user-admin → K8s API server | App Store admin populates `spec.secretRef` source Secret with raw credential material; operator reads it via kr8s |
| K8s API server → operator pod | kr8s reads raw Secret bytes into operator process memory; operator must never persist or log them |
| operator → log stream | Anything passed to `logger.*` may flow to kubectl logs, central logging, or audit systems — defense-in-depth: never include decoded key material |
