### Defects (Non-Critical)

1. **`blueprints.py` logger.warning TypeError** — `logger.warning(..., file=sys.stderr)` on line 60 will crash `scan_blueprints()` when encountering a malformed YAML file. Fix: remove the `file=sys.stderr` kwarg. Affects MCPS-04, MCPS-05.

2. **LOG_LEVEL env var not wired** — `config.py` declares it and README documents it, but `server.py` hardcodes `level=logging.INFO`. Fix: `level=getattr(logging, config.LOG_LEVEL, logging.INFO)`. Affects DEPLOY-03.

3. **PYTHONPATH missing from openclaw.json startup** — Container deployment is safe (Dockerfile handles it). Direct process spawn via openclaw.json will fail on `webapp.*` imports. Fix: add PYTHONPATH to startup env block. Affects AGNT-03.
