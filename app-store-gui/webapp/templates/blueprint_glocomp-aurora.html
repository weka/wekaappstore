Introduction

GC360-AURORA is Glocomp / Keppel's Autonomous Unified Response Operations and Real-time Analysis platform for disaster response and flood management. It combines IoT ingestion (flood gauges, weather stations, drone telemetry), NVIDIA NIM language and vision models, multi-modal RAG, and an H2O.ai flood intelligence agent into a single Kubernetes-native stack.

All AI workloads are backed by WEKA NeuralMesh AIDP through a dedicated aurora-weka-sc StorageClass, providing high-throughput parallel storage to Milvus, Triton, NIM model caches, Kafka, TimescaleDB, and the OpenMetadata governance catalog. Identity is centralised in Keycloak, observability runs through the Prometheus + Grafana stack with NVIDIA DCGM telemetry, and ingress is served by Envoy Gateway via Kubernetes Gateway API HTTPRoutes.
